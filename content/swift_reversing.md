Debugging swift in radare2
```
r2 -e bin.lang=swift -e bin.demanglecmd=true <program name>
```

| Command | Actions       |
| is      | List symbol   |
| iS      | List sections |
| ic      | List classes  |
| ~..     | less          |

```swift
import Foundation

// Checking condition
let test_number = 5

let condition: Bool
if test_number > 3 {
    condition = true
}
else {
    condition = false
}
```
